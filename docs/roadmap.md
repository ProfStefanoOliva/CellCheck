# CellCheck Roadmap

Questa roadmap e prudente e orientata a una crescita incrementale del progetto.

## Planned Milestones

- `v0.1`: project skeleton
- `v0.2`: data models
- `v0.3`: `.ccal` serialization
- `v0.4`: Excel color scanner
- `v0.5`: profile importer
- `v0.6`: correction engine
- `v0.7`: scoring
- `v0.8`: minimal GUI
- `v0.9`: report viewer
- `v1.0`: first usable release
