# CellCheck Branding Assets

Questa cartella contiene gli asset grafici sorgente usati dal progetto CellCheck.

## File attesi

- `cellcheck_logo_square.png`
- `cellcheck_logo_horizontal.png`
- `cellcheck_icon.ico`

## Scopo

- `cellcheck_logo_square.png`: logo quadrato per usi compatti o futuri pannelli/app launcher.
- `cellcheck_logo_horizontal.png`: logo orizzontale per documentazione o intestazioni future.
- `cellcheck_icon.ico`: icona applicativa usata dalla GUI PySide6 quando disponibile.

## Nota

Gli asset non vengono rigenerati dal codice del progetto. Il packaging dell'eseguibile e la distribuzione finale dell'icona saranno gestiti in una fase successiva.
